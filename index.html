My User Scripts

These are a few of the user scripts that I have written for Stack Overflow. They aren't all polished and most are incomplete. It should take about 6-8 weeks to finish them all. Bug reports are appreciated, feature requests are useful but don't expect anything to happen right away.

I will be expanding the descriptions here, as well as including help text where necessary. Styling improvements to come.

AJAX Listener

A simple AJAX listener that intercepts all AJAX requests made on a page and logs them to the console for review.

Enable Logging

Enables Stack Overflow's built in logging / debugging system.

Moderation Request Archiver

Provides an interface to archive formatted moderation requests in a given chat room. Must be a room owner or moderator of the current room for the interface to show up. This is an old version, the newest version is being maintained by the SOCVR

Moderation Request Generator

Provides an interface to automate the formatting and sending of moderation requests from a post to a given chat room. Currently only works for close vote requests on questions.

Trainwreck

Provides an interface to automate the formatting and sending of a message to the current chat room. Format is [#RekdTrain](link to random image of trainwreck).

Unclosed Request Review

Provides a button in the chat controls which navigates to a search page for close vote requests in the SOCVR. Once on the search page, the script removes any false positives, then makes an API query to determine which questions are closed, hides the closed questions, and provides some useful information for the open requests such as current close votes, score, and a link to the post in the case of improperly formatted URLs

Inline Editing Everywhere

Enables inline editing even if you don't have the inline editing privilege on a given site. Edits must still go through review if you don't have the editing privilege.

Sticky Vote Controls

Makes voting controls stick to the top of the page when scrolling down a long post.

Magic™ Editor

This was originally forked from The Stack Exchange Editor Toolkit, but has since been heavily modified by mostly myself and Mogsdad. We have included a massive rewrite of the RegEx matches and replacements, the reasoning logic, the extensiveness of the RegEx, the logic itself, a diff, and much more. More information to come.

Magic Tag Review

Provides a new review interface for reviewing questions in a specific tag based on close or delete votes. A new section is provided one the /review page. This will continue to exist but is going to be superseded by the new Magic Review Tool which will have many more features.

NATO Enhancements

Loads the full post and question as they would be seen in the wild with all controls enabled.

NATO Hide Questions

Hides the questions loaded by the NATO Enhancements script, this will eventually be merged as an option into the NATO Enhancements script.

Review Enhancements

Loads all posts as they would be seen in the wild with all controls enabled while in review.

Votes Page Enhancements

Loads posts as they would be seen in the wild with all controls enabled on your votes page. Useful for reviewing posts that you have voted on.

Thing Checker

A proof of concept audit detection script which intercepts the review AJAX requests and looks for the isAudit property of the returned JSON object. This is not intended to be used. If you use it, you run the risk of being manually review banned. A better option is to review to the best of your ability and report any bad audits on meta. A review ban incurred by multiple bad audits can be lifted by moderators if the bad audits are indeed bad audits.
